# 📊 Структура проекта EcoEats Bot

## 📁 Файловая структура

```
ecoeats_bot/
│
├── 📄 bot.py                    # Основной файл бота (простая версия)
│   └── Токен прямо в коде, строка 23
│
├── 📄 bot_with_env.py           # Версия с .env файлом (безопаснее)
│   └── Токен хранится в отдельном .env файле
│
├── 📄 requirements.txt          # Зависимости Python
│   ├── aiogram==3.4.1
│   └── python-dotenv==1.0.0
│
├── 📄 .env.example              # Пример файла с токеном
│   └── Скопируйте в .env и добавьте свой токен
│
├── 📄 README.md                 # Полная документация
│   └── Подробная инструкция по установке и использованию
│
├── 📄 QUICKSTART.md             # Быстрый старт за 5 минут
│   └── Краткая инструкция для нетерпеливых
│
└── 📄 PROJECT_STRUCTURE.md      # Этот файл
    └── Визуализация структуры проекта
```

1. **Экологичность** - основная ценность проекта

4. **Удобство** - все в одном мессенджере
